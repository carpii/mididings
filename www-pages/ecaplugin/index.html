About

ecaplugin.py is a tool to generate the unwieldy ecasound command lines for LADSPA and LV2 plugins from Ardour sessions or JACK Rack configurations.

Download

Repository at GitHub

Dependencies

  • Python (2 or 3)
  • Beautiful Soup 4 (python-bs4)

Usage

Run python ecaplugin.py --help for a full list of command line options.
